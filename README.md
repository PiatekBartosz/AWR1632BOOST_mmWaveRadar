# This repository contains code that interfaces with mmWave AWR1642

# Installation
Install all the dependencies from using conda:
```
conda env create -n <NewEnvName> --file environment.yml
```

# Run

## To run:
```
python plot_AWR1642_data.py 
```

## Options:
```
-c 'cfg_port_name' 
```

```
-d 'data_port_name'
```

```
-C 'cfg_name.cfg'
```
